# 프로젝트 소개

AI-Content-OS는 단순한 자동화 프로그램이 아니다.

AI를 활용하여

- 콘텐츠를 조사하고
- 정보를 분석하고
- 다양한 플랫폼에 맞게 재가공하고
- 수익을 창출하며
- 운영 데이터를 분석하여 개선하는

AI 기반 콘텐츠 운영 시스템(Content Operating System)을 구축하는 프로젝트이다.

최종 목표는 사람이 반복 작업을 하지 않아도 AI가 여러 비즈니스를 운영할 수 있는 환경을 만드는 것이다.

# 핵심 철학

## 1. Documentation First

문서를 먼저 작성한다.

설계가 끝난 후 코드를 작성한다.

---

## 2. Modular

모든 기능은 독립적인 모듈로 개발한다.

교체 가능해야 한다.

---

## 3. AI Native

AI가 수행 가능한 작업은 사람이 하지 않는다.

---

## 4. Reusable

중복 코드를 만들지 않는다.

중복 프롬프트도 만들지 않는다.

---

## 5. Scalable

Instagram만을 위한 프로젝트가 아니다.

향후 모든 플랫폼으로 확장 가능해야 한다.

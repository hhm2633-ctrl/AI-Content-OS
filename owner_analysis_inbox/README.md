# Owner Analysis Inbox

분석을 맡길 이미지, 영상, PDF, 문서, 링크 목록, 캡처 자료를 이 폴더에 넣습니다.

경로:

`C:\Users\가산 솔리드옴므\Documents\GitHub\AI-Content-OS\owner_analysis_inbox`

## 사용 방법

1. 자료를 이 폴더에 복사합니다.
2. 새 Codex 작업에서 `owner_analysis_inbox 분석해`처럼 지시합니다.
3. Codex는 먼저 파일명·크기·수정 시각만 확인하고 필요한 자료부터 점진적으로 읽습니다.
4. 이미지·영상 원본 전체를 채팅 문맥에 다시 싣지 않고 분석 결론만 프로젝트 학습 파일에 남깁니다.

## 보호 규칙

- 자료가 들어왔다는 이유만으로 자동 실행하지 않습니다.
- 원본은 명시적 승인 없이 이동·삭제·이름 변경하지 않습니다.
- 원자료 자체는 사실 확정, 성과 증거, 소유자 승인으로 간주하지 않습니다.
- 분석이 끝난 결론만 `knowledge/owner_feedback/`, `knowledge/owner_directives/`, 또는
  `docs/RESEARCH/`의 적절한 파일로 승격합니다.
- 긴 영상은 구간별로 점진 분석하고, 다음 작업에는 핵심 장면·카피·편집 규칙만 인계합니다.

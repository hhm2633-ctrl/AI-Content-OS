# AI-Content-OS Project Bible

## Vision
AI-Content-OS는 트렌드 수집부터 콘텐츠 생성, 이미지 생성, 카드뉴스 제작, 발행, 분석까지 자동화하는 AI 콘텐츠 운영체제(OS)이다.

## Goals
- 카드뉴스 자동화
- Shorts 자동화
- Blog 자동화
- Publishing 자동화
- Analytics 자동화

## Principles
- WorkflowEngine 유지
- workflow_completed 절대 유지
- Sprint 단위 개발
- Fallback First
- Cache First
- Modular Architecture

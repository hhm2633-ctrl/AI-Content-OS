# Competitor Engine

## Purpose

The Competitor Engine analyzes competing accounts, formats, services, and content patterns so AI-Content-OS can recommend better topics, hooks, layouts, and publishing actions.

## Collection Targets

- Instagram
- YouTube
- Shorts
- Blog
- Service UI

## Analysis Items

- Hook
- Pattern
- CTA
- Layout
- Format
- Brand DNA
- Performance signals

## Outputs

- Competitor profile
- Repeated content patterns
- Hook and CTA map
- Format comparison
- Gap analysis input for Audit Engine

## Current Status

Planning only. Implementation belongs in the roadmap after the card news MVP remains stable.

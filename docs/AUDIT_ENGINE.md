# Audit Engine

## Purpose

The Audit Engine compares my account against competitor accounts, identifies blind spots, and recommends practical content actions.

## Analysis Flow

```text
My Account Analysis
-> Competitor Comparison
-> Blind Spot Detection
-> Score Calculation
-> Recommended Actions
```

## Scores

- `hook_score`
- `brand_score`
- `content_score`
- `growth_score`
- `save_score`
- `share_score`
- `format_score`

## Outputs

- Audit summary
- Strengths
- Weaknesses
- Blind spots
- Priority recommendations

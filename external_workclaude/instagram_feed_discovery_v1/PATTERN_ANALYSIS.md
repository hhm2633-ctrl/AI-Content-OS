# Pattern Analysis

## 표본 한계 (먼저 읽을 것)
목표는 carousel/card-news 형식 게시물 15~20건 수집이었으나, 실제 개인화 피드(홈+탐색)를 축소된
저속 스크롤로 반복 관찰한 결과 확보된 건수는 **1건**이다. 탐색 피드가 약 40개 게시물 풀 내에서
반복 순환되었고(새 콘텐츠 미로드), 그 풀 안에서 carousel은 이 1건뿐이었다. 따라서 아래는 "패턴
군집화"라기보다 **단일 관찰 사례의 구조 기록 + 이 계정 피드의 형식 분포 자체에 대한 관찰**이다.
"반복 노출되고 공개 반응이 높은 카드뉴스 제작 방식"을 일반화하는 근거로 사용할 수 없다.

## 관찰 1: 피드 형식 분포 자체가 1차 발견
- 홈 피드: 개인 사진 캐러셀 중심 (카드뉴스 0건)
- 탐색 피드: 릴스(영상) 절대 우위, carousel 비중 매우 낮음 (약 40건 중 1건, 약 2.5%)
- 시사점: 이 계정의 알고리즘 프로필은 영상 소비 위주로 학습되어 있고, 카드뉴스형 콘텐츠는
  이 계정 개인화 맥락에서 노출 우선순위가 낮다. 즉 "카드뉴스가 이 계정에 잘 안 뜬다"는 사실 자체가
  분석 결과이며, 카드뉴스 제작 전략을 이 표본만으로 최적화하는 것은 표본 편향 위험이 크다.

## 관찰 2: 확보된 1건의 구조적 특징 (참고용)
- 표지: 인물 사진(감성/화제성 소재) + 굵은 텍스트 오버레이 훅 — 클릭 유도력이 강한 "인용형 반전
  헤드라인" 패턴
- 본문: 이미지 오버레이보다 캡션 텍스트에 서술 비중을 둔 "썰(포스트) 텍스트형" 구조 — 순수
  인포그래픽형 카드뉴스와는 다르게, 캡션을 길게 읽게 만드는 뉴스 큐레이션/여론 재구성 포맷
- 참여 유도: 명시적 CTA 버튼 없이 마지막 질문 문장으로 댓글 유도 — 소프트 CTA
- 팔로워 대비 매우 높은 좋아요 비율(약 59%)은 탐색 알고리즘을 통한 비팔로워 확산 가능성을 시사

## CN-006 비교 (마지막 단계)
`external_workclaude/content_portfolio_v1/TOP20_PRIORITY.md`의 CN-006(`미니멀리즘 시작하는 법`,
cardnews, P1)과 이번 관찰 사례를 비교한다.

| 항목 | CN-006 (미니멀리즘 시작하는 법) | 이번 관찰 사례 (zzalqueen 거제 이슈) |
|---|---|---|
| 주제 유형 | 지식/라이프스타일 에버그린 | 시의성 뉴스/화제 이슈 |
| 소재 성격 | 정보 전달형(방법론) | 여론·감정 유발형(공감/반전) |
| 훅 구조 | (CN-006 자체 카피는 별도 산출물에서 확정 필요 — 본 파일에서 원문 재생성하지 않음) | 인용형 반전 헤드라인, 인물 사진 배경 |
| 텍스트 구조 | 카드뉴스 표준: 슬라이드별 짧은 포인트 텍스트 예상 | 캡션 서술형 텍스트 비중이 큰 비표준 구조 |
| 참여 동력 | 실용정보 저장 목적 예상(저장 지표 중요) | 여론 공감/댓글 논쟁 유도가 중심, 저장보다 공유·댓글 지표 두드러짐 |

**시사점**: 이번 표본(뉴스 큐레이션형)은 CN-006(정보 전달형)과 콘텐츠 목적 자체가 다르다. 표본이
1건뿐이라 "이런 방식이 CN-006보다 낫다"는 결론은 낼 수 없으며, 유일하게 확인 가능한 유의미한
시사점은: **훅 문구를 인용형·반전형으로 구성하고 표지에 인물/감성 이미지를 쓰는 방식이 탐색
알고리즘 노출에 유리할 수 있다는 가설** 정도이며, 이는 n=1 관찰에 기반한 가설일 뿐 검증된 패턴이
아니다. CN-006 실행 시 참고 가능한 유일한 제안은 표지 훅 문구를 "인용/반전형 서술"로 실험해보는
정도이고, 카드 구조·CTA·슬라이드 수 등은 이번 표본에서 일반화할 근거가 부족하다.

## 종합 결론
- 목표(15~20건) 대비 실제 수집(1건)은 크게 미달했으며, 이는 실행 실패가 아니라 **이 계정의 개인화
  피드 자체가 카드뉴스 형식에 편향되어 있지 않다는 관찰 결과**로 해석해야 한다.
- 카드뉴스 제작 전략을 위한 표본으로 이 데이터를 확장 사용하려면, 별도의 카드뉴스 특화 계정을
  팔로우/탐색하는 계정이나 해시태그 기반 조사(사용자가 명시적으로 검색어 사용을 허용할 경우)가
  필요하다.

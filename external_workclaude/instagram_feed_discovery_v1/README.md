# Instagram Personalized Feed Discovery V1

## 목적
기존 생활형·미니멀리즘·체크리스트 검색어 제한 없이, 사용자 계정(`ho_hong_hongs`)에 실제로 개인화 추천되는
홈/탐색 피드를 위에서부터 관찰하여 carousel/card-news 형식 게시물의 구조를 수집·분석한다.
검색어 입력이나 분야 사전 선택 없이 "추천되는 그대로"를 관찰하는 것이 핵심 제약이다.

## 실행 방식 및 안전장치
- 브라우저 자동화(Claude in Chrome)로 로그인된 실제 계정의 홈 피드(`instagram.com/`)와 탐색 피드
  (`instagram.com/explore/`)를 스크롤하며 관찰만 수행.
- 좋아요·댓글·저장·팔로우·DM·게시 등 상호작용은 전 과정에서 **일체 수행하지 않음**.
- 대량 자동 스크롤이 Instagram 이용약관(자동 수집 금지)과 충돌해 계정 속도제한/경고로 이어질 리스크가
  있어, 사용자 확인 후 목표 수집량을 50개 → 15~20개로 축소하고 저속으로 진행.
- 결과적으로 자연 피드에서 carousel 형식 자체가 극히 희소하여(아래 참고) 최종 수집량은 1건에 그침 —
  목표 미달을 사용자에게 재확인 후, 축소된 실측 결과로 보고서를 확정.

## 핵심 발견 (요약)
이 계정의 개인화 피드는 carousel/card-news 형식에 최적화되어 있지 않다:
- **홈 피드**: 상위 노출은 지인의 개인 사진 게시물 위주(카드뉴스 형식 아님, 연구 대상에서 제외).
- **탐색 피드**: 약 40개 게시물 풀이 반복 순환되었고, 그중 **릴스(영상)가 절대다수**, carousel 형식은
  단 1건(`zzalqueen` 계정의 뉴스 재구성형 카드뉴스)만 확인됨. 추가 스크롤에도 새 게시물이 로드되지
  않고 동일 세트가 반복되어(가상 스크롤 윈도우 슬라이딩), 이 계정의 탐색 추천 풀 자체가 작고
  carousel 비중이 낮다는 결론에 도달함.

이는 **개인화된 1개 표본**이며, Instagram 전체 사용자나 카드뉴스 크리에이터 생태계 전반을 대표하지
않는다. 일반화 불가.

## 파일 구성
- `RAW_COLLECTION.md` — 피드 탐색 로그와 원자료(구조만 기록, 원문/이미지 비복제)
- `CLASSIFICATION.md` — 수집된 1건에 대한 항목별 구조 분류
- `PATTERN_ANALYSIS.md` — 패턴 분석(표본 n=1 한계 명시) 및 CN-006 비교(마지막 단계)

## 표본 크기 및 신뢰도 경고
**n=1**. 아래 분류·패턴 분석은 "이 계정에 노출된 특정 1건의 구조 기록"이며, 통계적 일반화나
"반복 노출·고반응 패턴"을 뒷받침할 근거로 사용해서는 안 된다. 참고용 단일 사례 관찰로만 활용할 것.

# Warning Pattern Prompt Guide

경고형 패턴: 흔한 실수와 위험을 먼저 보여주고, 왜 위험한지 이유를 설명한 뒤
해결 방법을 제시한다. 톤은 다급하지만 신뢰감 있게 유지하며, 근거 없는 공포
조장은 피한다.

구조 힌트:
- Hook: "___ 이렇게 하면 계정 망합니다", "___ 전에 꼭 보세요" 스타일
- Problem/Reason: 흔한 실수와 그것이 위험한 이유를 함께 설명
- Solution: 실수를 피하는 구체적인 행동
- CTA: 저장 유도

참고: benchmark/CONTENT_PATTERNS.md의 Warning Pattern, benchmark/HOOK_LIBRARY.md의
First 2 Seconds / 시선강탈형(attention) 항목.

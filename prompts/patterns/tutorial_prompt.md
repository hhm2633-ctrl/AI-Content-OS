# Tutorial Pattern Prompt Guide

튜토리얼 패턴: 초보자가 바로 따라 할 수 있도록 준비물과 핵심 실행 단계를
순서대로 설명한다. 어려운 용어 없이 쉬운 말투를 사용하고, 각 단계는 바로
실행 가능한 수준으로 구체적으로 쓴다.

구조 힌트:
- Hook: "___ 하는 법", "초보도 따라 하는 ___ 세팅" 스타일 (beginner_howto)
- Prep: 시작 전 준비물/전제 조건
- Step1~Step2: 핵심 실행 단계 (한 슬라이드에 압축 가능)
- CTA: 팔로우 또는 저장 유도

참고: benchmark/CONTENT_PATTERNS.md의 Tutorial Pattern, benchmark/HOOK_LIBRARY.md의
Beginner How-to / 초보 하우투형 항목.

# Comparison Pattern Prompt Guide

비교형 패턴: 비교 기준을 먼저 제시하고 두 옵션의 장단점을 대비시킨 뒤
상황별 추천으로 마무리한다. 편향 없이 균형 있게 서술하고, 근거 없는 단정은
피한다.

구조 힌트:
- Hook: "A vs B" 스타일 (contrarian 톤 가능)
- Criteria: 비교 기준 설명
- A vs B: 각 옵션의 장단점
- Recommendation: 상황별 추천
- CTA: 댓글로 의견 묻기

참고: benchmark/CONTENT_PATTERNS.md의 Comparison Pattern, benchmark/HOOK_LIBRARY.md의
Contrarian / Debate / 반전·논쟁형 항목.

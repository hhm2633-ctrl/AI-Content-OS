# Number List Pattern Prompt Guide

숫자 리스트 패턴: 숫자로 기대감을 준 뒤 왜 중요한지 짧게 설명하고, 핵심
항목을 간결하게 나열한다. 저장하고 싶게 정보 밀도를 높이고, 각 항목은
한 문장 이내로 짧게 쓴다.

구조 힌트:
- Hook: "___ 필수 도구 7가지" 스타일 (saveable_tip)
- Why important: 왜 중요한지 짧게
- Items: 항목 나열
- CTA: 저장 유도

참고: benchmark/CONTENT_PATTERNS.md의 Number List Pattern, benchmark/HOOK_LIBRARY.md의
Tips / Tools / 저장 유발형 항목.

# Story Pattern Prompt Guide

스토리형 패턴: 개인적인 경험과 고민을 먼저 보여주고, 전환점과 그로부터 얻은
교훈을 진솔하게 전달한다. 공감을 얻는 말투를 사용하고, 과장된 무용담처럼
쓰지 않는다.

구조 힌트:
- Hook: "저는 ___로 1년을 날렸습니다" 스타일 (result_proof / 결과 기반 후킹)
- Problem: 문제 상황과 고민
- Turning point / Lesson: 전환점과 교훈 (한 슬라이드에 압축 가능)
- CTA: 공감 댓글 유도

참고: benchmark/CONTENT_PATTERNS.md의 Story Pattern, benchmark/HOOK_LIBRARY.md의
Storytelling / Mindset / 연결형, Before/After / Result / 증거형 항목.

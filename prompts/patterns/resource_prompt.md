# Resource Pattern Prompt Guide

리소스 큐레이션 패턴: 저장을 유도하는 후킹 뒤 선정 기준을 설명하고, 실제
추천 리소스를 나열한다. 실용적이고 신뢰할 수 있는 톤을 사용하며, 확인되지
않은 정보를 단정적으로 제시하지 않는다.

구조 힌트:
- Hook: "무료 AI 사이트 20개" 스타일 (saveable_tip)
- Criteria: 선정 기준
- Resources: 추천 리소스 나열
- CTA: 댓글 키워드로 자료 신청 유도

참고: benchmark/CONTENT_PATTERNS.md의 Resource Curation Pattern, benchmark/HOOK_LIBRARY.md의
List / Curation / 리스트형 항목.

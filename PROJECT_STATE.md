# Notes For Any AI

이 프로젝트는 일반적인 자동화 프로젝트가 아니다.

AI 기반 콘텐츠 운영 플랫폼(AI Content Operating System)을 구축하는 프로젝트이다.

답변 시에는 항상

- 유지보수성
- 확장성
- 비용
- 안정성
- 운영성

을 함께 고려한다.

단기적인 구현보다 장기적인 프로젝트 품질을 우선한다.
